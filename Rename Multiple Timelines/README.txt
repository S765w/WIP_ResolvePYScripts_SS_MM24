Rename Multiple Timelines Script

This script is designed to automate the process of renaming multiple timelines in DaVinci Resolve. It allows the user to select timelines and enter new names for them.



How to Use 

Ensure DaVinci Resolve is open and a project is loaded before running the script.
Run the script in a Python environment using Visual Studio Code
Follow the prompts in the terminal window to select timelines and enter new names for them.
The script will prompt you to enter a new name for each timeline in the current project.
Type the new name and press Enter.
If you want to skip renaming a timeline, simply press Enter without typing anything.
